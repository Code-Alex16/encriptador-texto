ENCRIPTADOR DE TEXTO

encriptador de texto desarrollado con html, personalizado con css y agregando funcionalidades con Javascript ejemplo de uso:
ingresas una palabra o parrafo de un texto donde se solicita y escoger una de las opciones: ejemplo:
se ingreso la palabra gato, se dio click en el boton encriptar su resultado en pantalla sera: gaitober.
restricciones: no se permite caracteres especiales o con acentos (!,@,etc..), ademas de palabras en mayusculas.
